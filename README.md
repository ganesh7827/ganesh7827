## 🧠 Background

My career has spanned some unusual territory for a cybersecurity professional:

- **Cyber Police (India)** — Cybercrime investigations, OSINT/SOCMINT, anti-piracy operations
- **Trust & Safety (Yahoo EMEA)** — Account compromise, SOC escalations, e-crime
- **Critical National Infrastructure (EirGrid)** — Security Architecture, incident investigation
- **Financial Services (BNY)** — Confidential

That range — from law enforcement to fintech — shapes how I think about security. I've investigated real data theft. I've interviewed employees. I've built evidence chains for legal teams. And now I see those same threat patterns emerging inside AI agent pipelines, with no adequate tools to detect or investigate them.
